# 家庭账号分发与安装指引（Story 12.1）

本文档用于家庭内测首轮交付，目标是让家人可在无人陪同情况下完成安装与登录。

## 1. 交付包清单（维护者）

维护者分发时至少提供以下内容：

1. 最新 APK 下载方式（推荐使用固定下载入口）。
2. 对应版本号（便于反馈问题时定位）。
3. 家庭测试账号（邮箱）与临时密码（通过私密渠道单独发送）。
4. 本文档链接（安装与登录步骤）。

> 安全要求：不要把真实账号密码写入代码仓库或公开群聊。

## 2. 首次安装步骤（家人）

1. 首次安装请直接使用维护者发送的 APK 下载链接；若手机里已经装过旧版本，也可在 App 内通过 `更多 -> 下载更新包` 获取新版。
2. 下载 APK 后点击安装。
3. 如果系统提示“禁止安装未知应用/未知来源”，按手机引导临时允许后继续。
4. 安装完成后打开 App，进入登录页。

## 3. 登录方式

1. 在登录页输入维护者提供的邮箱和密码。
2. 点击“登录”后，成功会进入首页。
3. 若提示“邮箱或密码错误”，请先核对账号信息再重试。
4. 若提示“登录失败，请检查网络后重试”，先确认网络再重试。

## 4. 忘记密码处理

当前版本不提供自助找回密码入口。  
若忘记密码，请联系维护者执行重置并重新分发临时密码。

## 5. 常见问题（FAQ）

### Q1：安装失败

- 检查是否使用了与旧版本一致的包名和签名；
- 检查是否允许了未知来源安装；
- 检查手机存储空间是否足够。

### Q2：登录失败

- 先确认邮箱与密码是否与维护者分发信息一致；
- 再确认当前网络可用（切换 Wi-Fi/移动网络重试）。

### Q3：网络异常

- 先切换 Wi-Fi / 4G / 5G 后重新登录；
- 若仍失败，请将当前网络环境和报错现象反馈给维护者。

### Q4：升级后出现异常

反馈给维护者时请一并提供：

- 当前版本号
- 手机机型
- 网络环境（Wi-Fi / 4G / 5G）
- 复现步骤（尽量按 1、2、3 步描述）
- 截图或录屏（如方便）

## 6. 维护者回收反馈建议

建议维护者建立一个固定反馈格式（群公告或文档模板），至少包含：

- 发现时间
- 版本号
- 机型
- 网络环境
- 复现步骤
- 截图（可选）
